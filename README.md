# 🚖 OLA Ride Analytics Dashboard

A comprehensive Power BI dashboard built to analyze OLA ride booking data and provide actionable business insights through interactive visualizations.

---

## 📌 Project Overview

This project analyzes over **100,000 OLA ride bookings** and transforms raw booking data into meaningful business insights using **Power BI**.

The dashboard enables users to monitor booking trends, revenue performance, ride success rates, vehicle performance, cancellations, and customer behavior through interactive reports.

---

## 🎯 Objectives

- Analyze overall ride booking performance
- Monitor booking success and cancellation rates
- Compare vehicle types across different metrics
- Track daily booking and ride trends
- Evaluate revenue generated from successful bookings
- Build an interactive dashboard for business decision-making

---

## 📊 Dashboard Pages

### 1️⃣ Overall Dashboard

Provides a complete overview of business performance.

**KPIs**

- Total Booking Value
- Successful Rides
- Total Success Booking Value
- Total Distance Travelled

**Visualizations**

- Booking Status Distribution
- Daily Booking Trend
- Daily Distance Travelled
- Interactive Filters

---

### 2️⃣ Vehicle Type Dashboard

Analyzes the performance of each vehicle category.

**Visualizations**

- Top Vehicle Types by Ride Distance
- Booking Distribution by Vehicle Type
- Booking Status by Vehicle Type
- Revenue Contribution by Vehicle Type

---

## 📈 Key Insights

- Analyze booking distribution across vehicle categories.
- Identify the most frequently used vehicle types.
- Monitor ride trends over time.
- Compare successful and cancelled rides.
- Measure revenue contribution of each vehicle segment.
- Evaluate overall business performance using KPIs.
